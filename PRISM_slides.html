Motivation

Scenario

  • Model Merging: Finding the optimal $\alpha$ on a small model and transferring it to a large model.
  • Quantization: Compressing LLMs from bf16 $\to$ INT8, nf4, or Q2. How much does it break?
  • Continual Learning: Fine-tune on Task A, but the performance of Task B decays. Why?

The Core Problem

There is a lack of an analytical/diagnosis formulation to explain these empirical results. We rely on heuristics and expensive evaluations.

Idea

Proxy Model and Target Model

  • Target ($T$): The expensive, large, or full-precision model we care about.
  • Proxy ($P$): The cheap, small, or compressed model we actually compute on.

Is the Proxy Model ($P$) good enough to estimate the performance of the Target Model ($T$)?

Explainability

We don't just want a black-box predictor. We need a mathematical framework that provides Explainability.

If the Proxy fails to track the Target, we need to mathematically isolate why.

4 / 15

Requirement

1. Explainability

The formulation must decouple different sources of errors (e.g., feature mismatch vs. classifier mismatch) so we can diagnose failures.

2. Computable

The metrics must be rigorously calculable from empirical data, without requiring infinite samples or unobservable ground truths.

3. Efficiency

Evaluating the proxy's validity must be significantly cheaper than fully evaluating or training the target model.

4. Robustness

The theory must hold across different architectures, dimensionalities, and varying levels of geometric distortion.

Formulation

Methodology Choices

Analyze optimal point ($\alpha^*$)

Comparing parameters requires extremely strong assumptions because different models live in disjoint, non-comparable spaces.

Analyze Empirical Risk ($\Delta \mathcal{R}$)

We analyze the gap: $\Delta \mathcal{R} = |\mathcal{R}_T - \mathcal{R}_P|$

Why? Risk is a unified scalar. If we can prove the empirical risk landscapes are geometrically bound to each other, their optimal points must be functionally close.

Modern Models

Feature Representation & Linear Projection

We express any model as a composition: $f(x) = h(\phi(x))$

  • $\phi(x)$: Non-linear feature representation backbone (maps to $Z$).
  • $h(z)$: Linear prediction head.

Math: The Derivation Logic

出發點 (Starting Point)

We want to bound the difference in performance between Target ($T$) and Proxy ($P$):

$|\mathcal{R}_T(Z_T) - \mathcal{R}_P(Z_P)|$

橋樑 (Bridge: Linear Alignment)

Introduce an optimal linear isometric map $W$ to project Proxy features to Target space ($Z_P W$).

三角不等式 (Triangle Inequality)

We decompose the total risk gap into two components:

$$|\mathcal{R}_T(Z_T) - \mathcal{R}_P(Z_P)| \le \underbrace{|\mathcal{R}_T(Z_T) - \mathcal{R}_T(Z_P W)|}_{\text{Feature Error}} + \underbrace{|\mathcal{R}_T(Z_P W) - \mathcal{R}_P(Z_P)|}_{\text{Head Error}}$$

Prob (Lipschitz Bound)

Using Lipschitz constant $K_{feat}$, the Feature Error is bounded by the geometric distance: $K_{feat} \cdot \|Z_T - Z_P W\|_2$.

Math Result: Geometric Insights

By solving for the optimal $W$, we achieve an Exact Geometric Decomposition of the feature error using the Procrustes Similarity ($\Omega$):

$$\min_{W} \frac{1}{n} \| Z_T - Z_P W \|_F^2 = \underbrace{(\rho_T - \rho_P)^2}_{\text{Scale Mismatch}} + \underbrace{2\rho_T\rho_P(1 - \Omega(Z_T, Z_P))}_{\text{Shape Mismatch}}$$

Unified Risk Discrepancy Bound

$$|\mathcal{R}_T - \mathcal{R}_P| \le K_{feat} \sqrt{ (\Delta \rho)^2 + 2\rho_T\rho_P(1-\Omega) } + \text{Head Error}$$

Geometric Insights: The functional performance gap is strictly bounded by:

  • Scale ($\Delta \rho$): Do the feature norms match?
  • Shape ($\Omega$): Is the intrinsic topology of the manifolds isomorphic?
8 / 15

Math Discuss: Deep Dive

1. CKA vs. Procrustes ($\Omega$)

CKA uses the Frobenius norm, which provides a looser bound.

Our $\Omega$ relies on the Nuclear Norm ($\|Z_T^\top Z_P\|_*$). This guarantees an exact geometric decomposition for rotational alignment, strictly tightening the theoretical bound.

2. Head Bound

Head Error is bounded by: $\| \Sigma_P^{1/2} (W H_T - H_P) \|_F$

By incorporating the covariance matrix $\Sigma_P$, we project the error onto the active subspace of the data. Mismatches in "empty" directions don't penalize the bound.

3. $\rho$ (Scale)

$\rho$ represents the Root Mean Square (RMS) energy of the representations. Decoupling $\rho$ from $\Omega$ allows us to distinguish between a model that has structurally collapsed vs. one that is just numerically scaled down.

Quantization

In Quantization, we set Target = Full Precision ($F$) and Proxy = Quantized ($Q$).

The Identity Regime

Quantization adds numerical noise but doesn't intentionally rotate the latent space. Thus, the alignment map is the identity: $W = I$.

Head Stability

Modern quantization keeps the final classification head in high precision (FP16). Thus, $H_Q \approx H_F$, and the Head Error vanishes.

The Simplified Bound

$$|\mathcal{R}_F - \mathcal{R}_Q| \lessapprox K_{feat} \sqrt{ \underbrace{(\rho_F - \rho_Q)^2}_{\text{Scale Collapse}} + \underbrace{2\rho_F\rho_Q(1-\Omega)}_{\text{Feature Distortion}} }$$

We can estimate degradation purely from geometric statistics!

Quantization Experiments

Experimental Protocol

Rank different quantization methods (e.g., AWQ vs. GPTQ vs. NF4) purely by calculating $\Omega$ and $\Delta \rho$ on a small unlabeled calibration set.

Hypothesis

The geometric distortion score $\sqrt{(\Delta\rho)^2 + 2\rho_F\rho_Q(1-\Omega)}$ will strongly correlate with the actual downstream Perplexity/Zero-shot accuracy drop.

Detection

Aggressive quantization (e.g., INT2) will distinctly show Scale Collapse ($\rho_Q \ll \rho_F$) before severe structural collapse occurs.

Analysis

Worst Cases $\to$ Average Cases

Standard Lipschitz or Spectral bounds often assume the worst-case adversarial direction. By integrating the empirical data covariance $\Sigma_P^{1/2}$ into the Head Bound, PRISM shifts the analysis to the average-case behavior on the actual data manifold, making the bound practically tight.

Remove Scale

By extracting $\rho$ from the formulation, the Procrustes Similarity $\Omega \in [0, 1]$ becomes a strictly scale-invariant metric.

This is crucial: A feature vector that is simply halved in magnitude preserves its semantic meaning perfectly ($\Omega=1$), but standard distance metrics would flag it as a massive error. PRISM accurately isolates pure structural shape.
