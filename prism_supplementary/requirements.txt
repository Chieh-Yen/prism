#     Fix: use --no-build-isolation.

#   Issue B — pyproject.toml license field:
#     GPTQModel uses `license = "Apache-2.0"` (PEP 639 SPDX string).
#     setuptools < 71.0 rejects this format (only accepts {file/text: ...}).
#     Fix: upgrade setuptools >= 71 before installing GPTQModel.
#
# Recommended: use install.sh (handles all of this automatically)
#
#   bash install.sh
#
# Or manually:
#   Step 1 — torch (CUDA 12.8 wheel, must come first):
#     pip install torch==2.10.0 torchaudio==2.10.0 torchvision==0.25.0 \
#         --index-url https://download.pytorch.org/whl/cu128
#
#   Step 2 — upgrade setuptools (required by GPTQModel pyproject.toml):
#     pip install "setuptools>=71"
#
#   Step 3 — everything else:
#     pip install -r requirements.txt
#
#   Step 4 — GPTQModel (after torch + setuptools>=71 are present):
#     pip install GPTQModel==5.7.0 --no-build-isolation
